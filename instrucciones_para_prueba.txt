1- Instalar Python con y darle a la opción de agregar al PATH antes de la instalacion en la interfaz de instalación
	https://www.python.org/downloads/
2- Crear el proyecto desde la consola de Google cloud con la cuenta del cliente a ser utilizada
3- En APIs y servicios (en la cuenta del usuario a guardar los documentos) habilitar las APIs de Google drive y google sheets
4- En el apartado de credenciales, dar la opción y darle a ID de cliente de OAuth, luego elegir el tipo de aplicación 
como App de escritorio y dar acceso publico en la pantalla de consentimiento (descargar el archivo JSON de la credencial y 
cambiar el nombre a credentials.json)
5- Las credenciales agregar al directorio donde se encuentran los ficheros main.py, crear_documentos.py 
6- Crear una carpeta en el drive del cliente y HABILITAR A LA CARPETA ACCESO GENERAL COMO EDITOR PARA CUALQUIERA CON EL ENLACE,
en el link de la carpeta revisar el ID que se encuentra en la ruta de la web (https://drive.google.com/drive/folders/id_de_carpeta)
y asignarle ese id a la variable de tipo string PARENT_FOLDER_ID, luego guardar el codigo.
7- Ya instalado Python y descargados los ficheros, en cmd ir hasta el directorio y ejecutar el siguiente código
para instalar las librerías:
	pip install --upgrade google-api-python-client google-auth-oauthlib google-auth python-docx requests qrcode[pil] Pillow 
8- Y ahora ejecutar el siguiente código para correr el script:
	python main.py
